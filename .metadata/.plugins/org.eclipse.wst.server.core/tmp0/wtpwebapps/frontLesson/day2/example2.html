번호 주요개념 설명 예시
1 Color 색상은 미리 정의된 색상 이름 또는 RGB, HEX, HSL, RGBA, HSLA 값을 사용하여 지정됩니다. < h1 style="color:Tomato;">
Hello World </h1>
2 Border CSS 테두리 속성을 사용하면 요소 테두리의 스타일, 너비 및 색상을 지정할 수 있습니다. p.dotted {border-style: dotted;}
p.dashed {border-style: dashed;}
p.solid {border-style: solid;}
p.double {border-style: double;}
p.groove {border-style: groove;}
p.ridge {border-style: ridge;}
p.inset {border-style: inset;}
p.outset {border-style: outset;}
p.none {border-style: none;}
p.hidden {border-style: hidden;}
p.mix {border-style: dotted dashed solid double;}
3 Background CSS 배경 속성은 요소에 배경 효과를 추가하는 데 사용됩니다. div { background: rgba(0, 128, 0, 0.3)
/* Green background with 30% opacity */ }
4 Margins 여백은 정의된 테두리 외부의 요소 주위에 공간을 만드는 데 사용됩니다. p { margin-top: 100px; margin-bottom: 100px; margin-right: 150px; margin-left: 80px; }
5 Padding 패딩은 정의된 테두리 내부의 요소 콘텐츠 주위에 공간을 만드는 데 사용됩니다. div { padding-top: 50px; padding-right: 30px; padding-bottom: 50px; padding-left: 80px; }
6 height / width CSS height와 width속성은 요소의 높이와 너비를 설정하는 데 사용됩니다.
CSS max-width속성은 요소의 최대 너비를 설정하는 데 사용됩니다.
  1. auto- 기본값입니다. 브라우저는 높이와 너비를 계산합니다.
  2. length- 높이/너비를 px, cm 등으로 정의합니다.
  3. %- 컨테이닝 블록의 백분율로 높이/너비를 정의합니다.
  4. initial- 높이/너비를 기본값으로 설정합니다.
  5. inherit- 높이/너비는 부모 값에서 상속됩니다.
7 Box Model CSS에서 "상자 모델"이라는 용어는 디자인과 레이아웃을 말할 때 사용됩니다. CSS 상자 모델은 기본적으로 모든 HTML 요소를 감싸는 상자입니다.
여백, 테두리, 패딩 및 실제 콘텐츠로 구성됩니다.
  1. 콘텐츠 - 텍스트와 이미지가 표시되는 상자의 콘텐츠
  2. 패딩 - 콘텐츠 주변 영역을 지웁니다. 패딩은 투명합니다
  3. 테두리 - 패딩과 콘텐츠를 둘러싸는 테두리
  4. 여백 - 테두리 밖의 영역을 지웁니다. 여백이 투명하다
8 Outline 아웃라인은 요소의 경계 외부에 그려진 선입니다. 속성 outline-style은 윤곽선의 스타일을 지정하며 다음 값 중 하나를 가질 수 있습니다.
  • dotted- 점선 윤곽선을 정의합니다.
  • dashed- 점선 개요를 정의합니다.
  • solid- 솔리드 아웃라인을 정의합니다.
  • double- 이중 개요를 정의합니다.
  • groove- 3D 그루브 아웃라인 정의
  • ridge- 3D 능선 윤곽을 정의합니다.
  • inset- 3D 삽입 윤곽을 정의합니다.
  • outset- 3D 외곽선을 정의합니다.
  • none- 개요를 정의하지 않음
  • hidden- 숨겨진 윤곽을 정의합니다.
9 Text HTML 문서 내의 내용 및 기타 텍스트 요소를 가리킵니다. 다음과 같은 속성들을 가집니다.
  • Text Color
  • Text Alignment
  • Text Decoration
  • Text Transformation
  • Text Spacing
  • Text Shadow
10 Fonts "글꼴"은 웹 페이지에서 사용되는 텍스트의 모양을 지정하는데 사용되는 속성입니다 CSS에는 다섯 가지 일반 글꼴 모음이 있습니다.
  1. Serif 글꼴은 각 문자의 가장자리에 작은 획이있습니다. 그들은 격식과 우아함을 만듭니다.
  2. Sans-serif 글꼴에는 깔끔한 선이 ​​있습니다(작은 획이 첨부되지않음). 모던하고 미니멀한 룩을 연출합니다.
  3. 모노스페이스 글꼴 - 여기에서 모든 문자는 동일한 고정 폭을 가집니다. 그들은 기계적인 모습을 만듭니다.
  4. 필기체 글꼴은 사람의 손글씨를 모방합니다.
  5. 판타지 글꼴은 장식/재미있는 글꼴입니다.
11 Link CSS를 사용하면 다양한 방식으로 링크의 스타일을 지정할 수 있습니다. 네 가지 링크 상태는 다음과 같습니다.
  • a:link- 정상적인 미방문 링크
  • a:visited- 사용자가 방문한 링크
  • a:hover- 사용자가 마우스를 올려놓으면 링크
  • a:active- 클릭하는 순간 링크
12 List HTML에는 두 가지 주요 유형의 목록이 있습니다.
  • 정렬되지 않은 목록('ul') - 목록 항목은 글머리 기호로 표시됩니다.
  • 정렬된 목록('ol') - 목록 항목은 숫자 또는 문자로 표시됩니다.
 ul.a { list-style-type: circle; }
ul.b { list-style-type: square; }
ol.c { list-style-type: upper-roman; }
ol.d { list-style-type: lower-alpha; }
13 Table HTML 표의 모양은 CSS를 사용하여 크게 개선할 수 있습니다.
  • Table Borders
  • Table Size
  • Table Alignment
  • Table Style
  • Table Responsive
14 Display 속성은 display요소가 표시되는지 여부/방법을 지정합니다.
모든 HTML 요소에는 요소 유형에 따라 기본 표시 값이 있습니다.
대부분의 요소에 대한 기본 표시 값은 block또는 입니다 inline. 		li { display: inline; }
h1.hidden { display: none;}
h1.hidden { visibility: hidden; }
